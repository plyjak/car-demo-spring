# car-demo-spring
Demo spring app

Apka napisana w Java, backend do drugiego projektu. Build i run standardowo jak w każdym projekcie Gradle.

'Duże' użyte technologie/frameworki:
Java 17
Spring Boot
Hibernate
H2 in-memory

endpointy:

GET api/cars
GET api/cars/{id}
PUT api/cars/{id}
POST api/cars
DELETE api/cars/{id}
GET api/stats
